# Simple Todo
Projeto lista de tarefas simples desenvolvido com C#.
## Objetivo
O projeto tem o objetivo de aprimorar conceitos de desenvolvimento de softwere com padrões de projetos com foco na utilização do Entity FrameWork.

### Funcionalidades
O programa é capaz de listar as adicionar, consultar, atualizar e deletar as tarefas criadas.
Utilizando o banco de dados SQLITE e o Entity Framework para persistência das informações.

~~~
######################################################
                     Simple ToDo!
######################################################


1 - Tarefas Pendentes.
2 - Criar Tarefa.
3 - Atualizar Tarefa.
4 - Concluir Tarefa.
5 - Excluir Tarefa.
6 - Tarefas Concluídas.
0 - Sair.

Opção:

~~~
